# 🌐 Portfolio Full Stack

Portfolio personal desarrollado como proyecto Full Stack utilizando **Spring Boot** en backend y **React** en frontend.

El objetivo del proyecto es crear una web profesional donde mostrar mis proyectos, experiencia y habilidades como desarrolladora Full Stack, incluyendo gestión de contenido mediante un panel de administración.

---

## 🚀 Tecnologías utilizadas

### Backend

- Java 21
- Spring Boot 3.5.6
- Spring Data JPA
- Spring Security
- PostgreSQL
- Maven
- JWT (JSON Web Token)
- Lombok

### Frontend

- React
- Vite
- JavaScript
- SCSS
- React Router

### Herramientas

- Git
- GitHub
- Postman
- IntelliJ IDEA
- PostgreSQL / pgAdmin
